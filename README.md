
RX Serial Control v1
Esta es la primera versión de un programa de control para receptores.
Este código se utilizó como base para otros proyectos donde fue necesario controlar un receptor.
El código fue escrito para Borland Delphy 6 y 7.
Fue probado en Windows XP y 7.
